### バッファなしチャネル
```go
var c = make(chan int)
var a string

func f() {
    a = "Hello World"
    <-c
}

func main() {
    go f()
    c <- 0
    print(a)
}
```

このプログラムは "Hello World" が出ることが保証されている。
